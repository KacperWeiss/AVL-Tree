template<class T >
bool AVL_Tree< T >::deleteValue ( T  value)

Public method that removes node with selected value. First it validate if value which has to be removed is not null, then it validates if value exists in the tree. If value exists, then private method deleteValue(Node<T> node, T value) is called and it decrements nodeCount value:

  • if value does not exists in the tree, or value is null it returns false
  • if value exist in the tree it returns true once value is removed

    value- value to delete
